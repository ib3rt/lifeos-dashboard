## Navigation Shortcuts

### Tree Navigation

| Shortcut | Description |
|----------|-------------|
| `↑` | Move selection up one item |
| `↓` | Move selection down one item |
| `←` | Collapse folder or move to parent |
| `→` | Expand folder or move to first child |

### Quick Navigation

| Shortcut | Description |
|----------|-------------|
| `Home` | Jump to first item in tree |
| `End` | Jump to last item in tree |
| `Backspace` | Go up one level in hierarchy |

### Opening Files

| Shortcut | Description |
|----------|-------------|
| `Enter` | Open selected file |
| `Space` | Open selected file (alternative) |

---

## Action Shortcuts

### Search

| Shortcut | Description |
|----------|-------------|
| `Cmd+K` / `Ctrl+K` | Focus search input |
| `↑` / `↓` | Navigate search results |
| `Enter` | Open selected search result |
| `Esc` | Clear search and close results |

### Document Actions

| Shortcut | Description |
|----------|-------------|
| `Cmd+P` / `Ctrl+P` | Print current document |
| `Cmd+S` / `Ctrl+S` | Save current position as bookmark |
| `Cmd+D` / `Ctrl+D` | Toggle dark/light theme |

### Help

| Shortcut | Description |
|----------|-------------|
| `Cmd+/` / `Ctrl+/` | Open keyboard shortcuts modal |
| `Esc` | Close any open modal |

## Platform-Specific Notes

### macOS

- Use `Cmd` (⌘) key for all shortcuts
- `Cmd+K` = `⌘+K`
- `Cmd+P` = `⌘+P`
- `Cmd+S` = `⌘+S`
- `Cmd+D` = `⌘+D`
- `Cmd+/` = `⌘+/`

### Windows / Linux

- Use `Ctrl` key for all shortcuts
- `Ctrl+K` = Focus search
- `Ctrl+P` = Print
- `Ctrl+S` = Save bookmark
- `Ctrl+D` = Toggle theme
- `Ctrl+/` = Show shortcuts

## Remembering Shortcuts

### By Category

- **Navigation**: Arrow keys, Home, End, Backspace
- **Search**: K (for "Kommand search")
- **Help**: / (question mark neighbor on some keyboards)
- **Print**: P (standard print key)
- **Save**: S (standard save key)
- **Theme**: D (for "Dark mode")

### Mnemonics

1. **K** → **K**ommand search bar
2. **P** → **P**rint document
3. **S** → **S**ave/bookmark position
4. **D** → **D**ark mode toggle
5. **/** → **/** for help/shortcuts

## Troubleshooting Shortcuts

### Shortcuts Not Working?

1. **Check focus**: Make sure you're not in an input field
2. **Check browser**: Some shortcuts may be browser-specific
3. **Check OS**: System shortcuts take priority
4. **Try alternative**: Use the mouse as backup

### macOS Issues?

- If `Cmd+/` doesn't work, try `Cmd+Shift+/`
- Some browsers use `Cmd+,` for preferences

### Windows/Linux Issues?

- If `Ctrl+K` doesn't focus search, click the search bar manually
- `Ctrl+/` may conflict with screen reader shortcuts
