# Keyboard Shortcuts Reference

## Navigation

| Shortcut | Action | Description |
|----------|--------|-------------|
| `↑` | Navigate Up | Move selection up in the file tree |
| `↓` | Navigate Down | Move selection down in the file tree |
| `←` | Collapse | Collapse the current folder |
| `→` | Expand | Expand the current folder |
| `Enter` | Open | Open the selected file or folder |
| `Backspace` | Go Up | Navigate to parent folder |
| `Home` | First Item | Jump to first item in tree |
| `End` | Last Item | Jump to last item in tree |

## Search & Actions

| Shortcut | Action | Description |
|----------|--------|-------------|
| `⌘K` / `Ctrl+K` | Focus Search | Focus the search input field |
| `⌘/` / `Ctrl+/` | Show Shortcuts | Display keyboard shortcuts modal |
| `Esc` | Clear/Escape | Clear search or close modal |

## File Operations

| Shortcut | Action | Description |
|----------|--------|-------------|
| `⌘P` / `Ctrl+P` | Print | Open browser print dialog |
| `⌘S` / `Ctrl+S` | Save Bookmark | Save current position as bookmark |

## Appearance

| Shortcut | Action | Description |
|----------|--------|-------------|
| `⌘D` / `Ctrl+D` | Toggle Theme | Switch between light and dark mode |

## Accessibility

- `Tab` - Move focus to next interactive element
- `Shift+Tab` - Move focus to previous element
- `Space` - Toggle expand/collapse on focused items
- `Enter` - Activate links and buttons

## Tips

1. **Quick Search**: Press `⌘K` to instantly focus the search bar
2. **Navigation**: Use arrow keys to navigate the file tree without mouse
3. **Deep Links**: Directly navigate to any document using URL hash (e.g., `#workspace/dashboard/README.md`)
4. **Recent Files**: Your 10 most recent files are automatically saved and accessible from the sidebar
5. **State Persistence**: Your theme preference, sidebar state, and expanded folders are saved automatically
