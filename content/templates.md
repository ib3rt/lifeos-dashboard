# Content Templates

## Marketing Copy Templates

### 1. Email Subject Lines
```
Pattern: [Power Word] + [Benefit] + [Urgency]
Example: "Exclusive: Double Your Productivity in Just 30 Days"
```

### 2. Blog Post Structure
```
1. Hook (first 100 words)
2. Problem statement
3. Solution introduction
4. Main points (H2s with subpoints)
5. Case study/examples
6. How-to steps
7. Common mistakes to avoid
8. Conclusion + CTA
```

### 3. Social Media Posts

**Twitter/X:**
```
Hook (1-2 sentences)
Value proposition
Link/CTA
Tags (3-5 relevant)
```

**LinkedIn:**
```
Professional insight
Personal experience tie-in
Key takeaways (bullet points)
Question/prompt for engagement
Call to action
```

**Instagram:**
```
Caption hook
Value/education/entertainment
Storytelling element
CTA (save/share/comment)
Line breaks for readability
```

### 4. Video Script Template
```
00:00 - Hook (attention grabber)
00:30 - Promise (what they'll learn)
01:00 - Bio/credibility
02:00 - Main content point 1
05:00 - Main content point 2
08:00 - Main content point 3
10:00 - Summary + key takeaways
11:00 - CTA (subscribe/like/comment)
```

### 5. Newsletter Template
```
Header: Issue #X - [Theme]
Greeting + personal note
Feature article (main value)
Quick tips section
Community highlight
Upcoming events
Footer + unsubscribe
```

### 6. Case Study Template
```
Challenge
Solution
Results
Key metrics
Testimonial
Lessons learned
```

## CTA Templates

| Situation | CTA Copy |
|-----------|----------|
| Newsletter | "Subscribe for weekly insights" |
| Blog | "Get the free checklist" |
| Video | "Subscribe for more tips" |
| Social | "Comment below with your thoughts" |

## Template Variables

```
{{TITLE}} - Content title
{{AUTHOR}} - Author name
{{DATE}} - Publication date
{{KEYWORD}} - Target SEO keyword
{{CTA_URL}} - CTA link
{{COMPANY}} - Company name
{{AUDIENCE}} - Target audience
```
