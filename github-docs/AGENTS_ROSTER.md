## Deployment Patterns

### Single Agent Tasks
Most requests route to one specialist based on domain.

### Multi-Agent Coordination
For complex projects, use coordinator pattern:
1. **Executive Support** or **Operations Coordinator** receives request
2. Breaks into subtasks for appropriate specialists
3. Aggregates outputs and resolves conflicts
4. Delivers unified response

### Emergency Escalation
Certain triggers immediately escalate to user:
- Security incidents ‚Üí Cybersecurity Guardian + immediate user alert
- Medical emergencies ‚Üí Health Coach + immediate user alert
- Legal issues ‚Üí Legal Advisor + immediate user alert
- Financial crises ‚Üí Finance Director + immediate user alert
