# February 2, 2026

## Planning Day

Spent most of today sketching out the architecture for the Second Brain app. The goal is to create something that feels both powerful and effortless.

## Design Decisions

### Why Obsidian + Linear?

**From Obsidian:**
- Local-first Markdown files
- Wiki-style linking [[like this]]
- Graph view of connections
- Plugin ecosystem concept

**From Linear:**
- Smooth, purposeful animations
- Keyboard-first interactions
- Clean, minimal aesthetic
- Fast, responsive UI

### Tech Stack

- **Next.js 14+**: App Router, Server Components
- **TypeScript**: Type safety throughout
- **Tailwind CSS**: Utility-first styling
- **gray-matter**: Frontmatter parsing
- **unified/remark**: Markdown processing
