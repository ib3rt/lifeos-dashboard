# Getting Started with Second Brain

Welcome to your **Second Brain** - a personal knowledge management system that combines the best of [[Obsidian]]'s linked thinking with [[Linear]]'s polished user interface.

## What is Second Brain?

A Second Brain is a methodology for saving and systematically reminding us of the ideas, insights, and connections we've gained through experience. It's an external digital repository for the things you learn and the resources you encounter.

## Creating New Documents

To create a new document:

1. Create a `.md` file in the appropriate folder
2. Add YAML frontmatter at the top:

```yaml
---
title: "Your Document Title"
date: "2026-02-03"
tags: ["tag1", "tag2"]
category: "concepts"
description: "Brief description of the document"
---
```

3. Write your content using Markdown
