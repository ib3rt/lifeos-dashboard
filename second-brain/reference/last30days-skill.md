# last30days-skill

## Repository
https://github.com/mvanhorn/last30days-skill

## Description
Claude Code skill that researches any topic across Reddit + X from the last 30 days, then writes copy-paste-ready prompts.

## What It Does
- Researches topics across Reddit and X from last 30 days
- Finds what community is upvoting and sharing
- Writes prompts that work today (not 6 months ago)
- Best for: prompt research, trending topics, tool best practices

## Usage Examples
- `/last30days prompting techniques for ChatGPT`
- `/last30days iOS app mockups`
- `/last30days What are the best rap songs lately`

## Installation (Tonight)
1. Clone repo to `~/.claude/skills/last30days`
2. Add API keys (OPENAI_API_KEY, XAI_API_KEY)
3. Test with `/last30days [topic]`
