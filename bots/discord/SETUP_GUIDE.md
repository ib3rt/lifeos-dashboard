# Discord Bot Setup - Quick Guide

## Step 1: Get Bot Token (2 minutes)

1. Go to: https://discord.com/developers/applications
2. Click **"New Application"**
3. Name it: **"Life OS Bot"**
4. Go to **"Bot"** tab on left
5. Click **"Add Bot"** → **"Yes, do it!"**
6. Under **TOKEN**, click **"Copy"**
7. **Paste token here**

⚠️ **KEEP TOKEN SECRET** — don't share it publicly

## Step 2: Invite Bot to Server (30 seconds)

1. Go to **"OAuth2"** → **"URL Generator"**
2. Select scopes:
   - ☑️ **bot**
   - ☑️ **applications.commands**
3. Select bot permissions:
   - ☑️ **Administrator** (or minimum: Send Messages, Read Messages, Embed Links)
4. Copy the generated URL
5. Open URL in browser
6. Select your server: **"Life OS Command Center"**
7. Click **"Authorize"**
